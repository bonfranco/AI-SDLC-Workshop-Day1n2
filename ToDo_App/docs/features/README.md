# Feature Implementation Map

This scaffold includes route and library placeholders for the following planned features:

1. Authentication (WebAuthn)
2. Todo CRUD with priority and due dates
3. Recurring todos
4. Reminders and notifications
5. Subtasks and progress
6. Tags and categories
7. Templates
8. Search and advanced filters
9. Export/import
10. Calendar view
11. Dark mode

Use the files in app/api and lib as implementation entry points.
